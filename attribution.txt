==========
Pocket PDB
==========

The pocket PDB was obtained from the Meeko GitHub repository, itself obtained
from the PDB page 1IEP.
https://github.com/forlilab/Meeko/blob/develop/example/tutorial1/input_files/1iep_protein.pdb

===========
Pocket JSON
===========

The cleaned pocket PDB was analyzed for pockets using the RowanSci
implementation of Pocketeer. The pocket JSON was exported from the RowanSci
platform.
